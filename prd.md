# 社区服务商城小程序 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景

本项目旨在构建一个社区服务商城微信小程序，整合**商城购物**与**上门服务**两大核心业务，为社区居民提供便捷的生活服务体验。用户可在小程序中购买各类商品，也可预约家政保洁、家电维修等上门服务，实现一站式社区生活服务。

### 1.2 项目目标

- 为用户提供丰富的商品购买服务
- 为用户提供便捷的上门服务预约体验
- 建立用户与服务提供者之间的高效沟通渠道
- 为商家提供商品管理和服务订单管理工具

### 1.3 适用范围

本产品适用于社区居民、服务商家、商品供应商及平台管理员。

---

## 2. 角色定义

### 2.1 用户端

指通过小程序购买商品、预约上门服务的社区居民。

### 2.2 商家端

指通过小程序管理商品、服务订单的商家和服务提供者。

### 2.3 管理员端

指通过小程序对平台商家和服务进行管理、审核的系统管理人员。

## 3. 用户端功能模块

### 3.1 首页

#### 3.1.1 搜索功能

- 支持关键词搜索商品和服务
- 搜索历史记录
- 热门搜索推荐

#### 3.1.2 轮播广告

- 展示促销活动、新品上架、服务优惠等信息
- 自动轮播，支持手动切换

#### 3.1.3 服务分类入口

以图标形式展示八大上门服务分类：

| 图标 | 分类名称 | 说明 |
| :--- | :--- | :--- |
| 🧹 | 家政保洁 | 日常保洁、深度清洁、开荒保洁 |
| 🔧 | 家电维修 | 空调维修、冰箱维修、洗衣机维修 |
| 🛠️ | 管道疏通 | 下水道疏通、马桶疏通、地漏疏通 |
| 🔑 | 开锁换锁 | 开锁服务、换锁芯、指纹锁安装 |
| 🚚 | 搬家服务 | 同城搬家、家具拆装、长途搬家 |
| 🧼 | 清洗服务 | 油烟机清洗、空调清洗、窗帘清洗 |
| 🏠 | 装修维修 | 墙面修补、水电维修、瓷砖修复 |
| 📦 | 上门安装 | 家具安装、家电安装、灯具安装 |

#### 3.1.4 商品推荐

- 展示热销商品
- 商品卡片包含图片、名称、价格
- 支持点击进入商品详情页

### 3.2 分类页

#### 3.2.1 分类导航

分为两大板块：

**上门服务分类**：
- 家政保洁（日常保洁、深度清洁、开荒保洁）
- 家电维修（空调维修、冰箱维修、洗衣机维修）
- 管道疏通（下水道疏通、马桶疏通、地漏疏通）
- 开锁换锁（开锁服务、换锁芯、指纹锁安装）
- 搬家服务（同城搬家、家具拆装、长途搬家）
- 清洗服务（油烟机清洗、空调清洗、窗帘清洗）
- 装修维修（墙面修补、水电维修、瓷砖修复）
- 上门安装（家具安装、家电安装、灯具安装）

**商城商品分类**：
- 日用百货（家居用品、清洁用品、厨房用品）
- 数码配件（手机配件、电脑配件、智能设备）
- 食品饮料（零食小吃、饮料酒水、生鲜果蔬）
- 母婴用品（婴儿食品、纸尿裤、玩具）
- 美妆护肤（护肤品、化妆品、个人护理）
- 服装鞋帽（男装、女装、鞋靴）
- 运动户外（运动器材、户外装备）

#### 3.2.2 分类筛选

- 点击分类标签查看该分类下的服务/商品列表
- 支持二级分类筛选

### 3.3 商品/服务详情页

#### 3.3.1 商品详情

- 商品图片轮播
- 商品名称、价格、规格、库存
- 商品描述、参数详情
- 加入购物车、立即购买

#### 3.3.2 服务详情

- 服务图片展示
- 服务名称、价格、时长
- 服务描述、服务范围
- 预约时间、预约人数
- 立即预约

### 3.4 购物车

- 商品列表展示（图片、名称、价格、数量）
- 数量增减、删除商品
- 商品总价计算
- 结算功能

### 3.5 订单管理

#### 3.5.1 商品订单

- 订单列表（待付款、待发货、待收货、已完成、退货中、退款中）
- 订单详情查看
- 取消订单、确认收货
- 订单评价

#### 3.5.2 服务订单

- 订单列表（待确认、待服务、服务中、已完成、已取消）
- 订单详情查看
- 取消订单、确认服务完成
- 服务评价

### 3.6 退货退款

#### 3.6.1 申请退货退款

- 在已完成或待收货订单详情页发起退货退款申请
- 选择退货原因（商品质量问题、与描述不符、不喜欢/不想买、其他）
- 填写退货说明
- 上传商品照片作为凭证
- 提交申请

#### 3.6.2 退货退款进度查看

- 订单列表显示退货/退款状态
- 订单详情页展示退货退款进度（待商家审核、商家同意、用户退货中、商家收货、退款处理中、退款完成）
- 支持查看退货地址

#### 3.6.3 与商家沟通

- 在订单详情页发起与商家的在线沟通
- 支持文字、图片消息
- 沟通记录保存，方便后续查看

### 3.7 消息中心

- 退货退款进度通知
- 商家回复消息通知
- 系统通知

### 3.8 个人中心

- 用户信息（头像、昵称、手机号）
- 订单入口（商品订单、服务订单）
- 优惠券、积分
- 地址管理
- 意见反馈
- 设置

## 4. 商家端功能模块

### 4.1 工作台

- 商家信息展示
- 数据统计（订单数量、销售额、服务完成率）
- 快捷操作（发布商品、发布服务、订单管理）

### 4.2 商品管理

- 商品列表（上架、下架状态）
- 添加商品（商品信息、图片上传、价格设置）
- 编辑商品、删除商品
- 库存管理

### 4.3 服务管理

- 服务列表（上架、下架状态）
- 添加服务（服务信息、图片上传、价格设置）
- 编辑服务、删除服务
- 服务分类管理

### 4.4 订单管理

#### 4.4.1 商品订单

- 订单列表（待发货、待收货、退货中、退款中）
- 发货操作
- 订单详情查看

#### 4.4.2 服务订单

- 订单列表（待确认、待服务、服务中）
- 确认订单、开始服务、完成服务
- 订单详情查看

### 4.5 退货退款管理

#### 4.5.1 退货退款审核

- 退货退款申请列表（待审核、已同意、已驳回）
- 查看退货申请详情（退货原因、说明、凭证照片）
- 审核操作（同意退货、驳回申请）
- 填写审核意见

#### 4.5.2 退货确认

- 用户退货后，商家确认收货
- 查看退货商品照片
- 确认无误后发起退款

#### 4.5.3 与用户沟通

- 在退货详情页发起与用户的在线沟通
- 支持文字、图片消息
- 沟通记录保存

### 4.6 个人中心

- 商家信息管理
- 店铺设置
- 提现管理

## 6. 业务流程

### 6.1 商品购买流程

用户进入首页 → 浏览/搜索商品 → 查看商品详情 → 加入购物车/立即购买 → 确认订单 → 支付 → 商家发货 → 用户收货 → 评价

### 6.2 服务预约流程

用户进入首页 → 选择服务分类 → 查看服务详情 → 选择时间/人数 → 确认订单 → 支付 → 商家确认 → 上门服务 → 用户确认完成 → 评价

### 6.3 商家审核流程

商家注册 → 提交资质材料 → 管理员审核 → 审核通过/驳回 → 商家入驻成功

### 6.4 退货退款流程

用户发起退货申请 → 选择退货原因 → 上传凭证 → 商家审核 → 商家同意/驳回 → 商家同意后用户退货 → 商家确认收货 → 发起退款 → 退款完成

## 8. 非功能性需求

### 8.1 性能要求

- 页面加载时间不超过3秒
- 搜索响应时间不超过1秒
- 订单提交响应时间不超过2秒

### 8.2 可用性要求

- 界面设计简洁直观，符合微信小程序设计规范
- 操作流程简单明了，减少用户学习成本
- 支持常见手机屏幕尺寸适配

### 8.3 安全性要求

- 用户和商家身份需进行验证
- 用户隐私信息需妥善保护
- 支付过程需确保安全可靠
- 管理员操作需记录日志
